Built for enthusiasts who care about details.

The application is designed to handle hierarchical data in a clean and efficient way — from brands, through models and generations, down to specific versions and configurations.

With a modern tech stack and scalable structure, it provides a solid foundation for expanding into a full-featured automotive database.

About this project

This project was created to combine modern frontend and backend technologies into a single, cohesive application.

Built with Angular and Spring Boot, it focuses on presenting structured automotive data in a clean and intuitive way.
